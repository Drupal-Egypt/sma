Social Mobile App (ver. 0.1)

Social Mobile App is an example application created by Vadim Mirgorod to demonstrate Backbone.js and jQueryMobile capabilities, it communicates with RESTful service written on Python. You can download recent version of both client and server side from github project page.

Technology stack

Client side

  • Backbone.js
  • jQuqeryMobile
  • iOS inpisred jQUery Mobile Theme
  • Glyphish icons by Joseph Wain
  • PhoneGap

Server side

  • CherryPy
  • MomPy
  • MongoDB
